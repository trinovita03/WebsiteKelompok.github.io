Study Courses

In the course of my studies in this University program, there are several courses that have catched my interest and motivated me to continue developing myself, as well as preparing for further advancement. Some of these courses are:

1. Operating System

"The Operating System" is a course that discusses the concepts, principles, and functions of computer operating systems. It covers various topics such as memory management, process management, file management, as well as the interaction between hardware and software in a computer system.

Example: Managing operations to ensure data integrity and efficient use of storage space. Interaction between Hardware and Software, helps users navigate their devices, install software, manage files, and troubleshoot common issues.

2. Database Systems

"Database Systems" is a course that discusses database concepts, design, implementation and management. The material taught includes data structures, query languages, data modeling, normalization, and data storage and retrieval techniques. The main goal is to understand how data is stored, organized, and accessed efficiently in a computer system.

Example: Database Administration, Application Development, Big Data Analysis Business Analytics such as using skills in retrieving data from databases, understanding relationships between data, and creating reports or data visualizations to provide valuable insights to management.

3. Embedded System

The "Embedded Systems course" discusses the design, development and implementation of embedded systems consisting of hardware and software embedded in electronic devices. The main focus is on systems designed to perform specific tasks in a constrained environment, such as microcontrollers, sensors, and actuators. We learn the basic concepts, hardware, programming languages, and related technologies necessary to design and program embedded systems.

Example: In industrial environments, embedded systems are used to control and automate production and manufacturing processes. also relevant in the development of IoT, where devices are connected wirelessly to exchange data and provide intelligent services. In the automotive industry, embedded systems are found in vehicles to control systems such as fuel injection, antilock brakes (ABS), and navigation.

4. Computer Network

The "Computer Network" course discusses the concepts, principles, and technology used in data communication between computers and other devices over networks. Students study topics such as network architecture, communications protocols, wireless technology, network security, and network systems administration. The focus is on understanding how data is transmitted, routed, and managed in computer networks.

Example: Designing, implementing, and maintaining computer networks in various organizations. We will use the knowledge gained from this course to install and configure network hardware and software, as well as to troubleshoot network connection and performance problems. Also identify and protect systems from security threats such as malware attacks, hacking, and data theft. Also implement security technologies such as firewalls, VPNs, and intrusion detection to protect networks and sensitive data.
